# vue-press-test

> 体验下vue-press，记录下自己的配置

## 安装

``` bash
# 安装依赖
`yarn add`

# dev运行
`npx vuepress dev docs` or `yarn run docs:dev`

# 会在docs/.vuepress下生成dist文件夹
`npx vuepress build docs` or  `yarn run docs:build`

```

## 查看配置
配置都在docs文件夹里面, 你可以结合[官方文档](https://vuepress.vuejs.org/zh/)一起看，效果更好哦！
